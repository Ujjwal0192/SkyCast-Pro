# SkyCast-Pro -  Modern Weather Application

SkyCast Pro is a responsive, single-page weather application built using React, Vite, Tailwind CSS v4, and the OpenWeather REST API. It provides real-time weather information for any city worldwide through a clean, modern, and user-friendly interface.

The app allows users to search for cities and instantly view current temperature, weather conditions, humidity, wind speed, pressure, sunrise and sunset times, along with dynamic weather icons. It features smooth UI interactions, loading states, and error handling to ensure a seamless user experience.

Designed with a professional dashboard layout and optimized performance, this project demonstrates efficient API integration, component-based architecture, and modern frontend development practices.
